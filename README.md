# Quick Guide

- Install bundler

  ```sh
  gem install bundler
  ```

- Install dependencies

  ```sh
  bundle install
  ```

- Compile style sheets

  ```sh
  bundle exec compass compile
  ```

- Watch changes in style sheets

  ```sh
  bundle exec compass watch
  ```

- Watch site changes

  ```sh
  bundle exec jekyll serve -w
  ```

  The site will be available at http://127.0.0.1:4000/
